Embed

A Tag Embed incorpora um conteúdo externo e tem como atibutos height, src, type, width e os atributos globais

Saiba mais

Exemplo:

Código:
<embed type="video/webm" src="https://www.youtube.com/embed/0eWUhXPhIaE?si=nJXs8L4dM143nYbA" width="200" height="200">

Object

A Tag Object incorpora um conteúdo externo e tem como atibutos data, form, height, name, type e width

Saiba mais

Exemplo:

Código:
<object data="https://img.freepik.com/psd-gratuitas/livros-empilhados-isolados-sobre-um-fundo-transparente_191095-17333.jpg?size=338&ext=jpg&ga=GA1.1.2008272138.1721174400&semt=ais_user" width="200" height="200">

Iframe

A Tag Iframe incorpora um conteúdo externo e tem como atibutos os atributos globais, allow, height, loading, entre outros

Saiba mais

Exemplo:

Código:
<iframe src="https://www.youtube.com/embed/gx-fg-dMROg?si=Vb8KEHwuTg85VypT" title="Linkin Park - Crawling"></iframe>
